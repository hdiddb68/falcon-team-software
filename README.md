## 🔥 О проекте

**Falcon Team** — набор OSINT-инструментов для проверки информации из открытых источников.

✅ **Falcon Team PC** — терминал для проверки email, телефона, IP, username  
✅ **Falcon Assistant** — голосовой помощник (Джарвис) для управления ПК  
✅ **Falcon Team Mobile** — офлайн-версия для Android

## 📦 Установка

```bash
# Клонируй репозиторий
git clone https://github.com/hdiddb68/falcon-team-software.git
cd falcon-team-software

# Установи зависимости
pip install -r requirements.txt

# Запусти Falcon Team PC
python falcon_pc_final.py

# Или запусти Джарвиса
python falcon_assistant.py



🎯 Команды (Falcon Team PC)
Команда	Пример	Что делает
email	email test@mail.ru	Проверка утечек (HIBP)
phone	phone 79161234567	Поиск по номеру (40+ источников)
ip	ip 8.8.8.8	Геолокация IP
user	user username	Поиск по username (VK, Telegram, GitHub...)
domain	domain google.com	WHOIS домена
report	report email test@mail.ru	Сохранить отчёт в JSON
🤖 Джарвис (Falcon Assistant)
Команда	Что делает
браузер	Открывает Google
ютуб	Открывает YouTube
музыка	Включает музыку
фалькон	Запускает Falcon Team PC
ноутбук	Открывает «Этот компьютер»
файлов	Открывает проводник
диск	Открывает диск C:
документы	Открывает папку «Документы»
загрузки	Открывает папку «Загрузки»
изображения	Открывает папку «Изображения»
рабочий стол	Открывает рабочий стол
время	Говорит текущее время
статус	Показывает загрузку CPU/RAM
выключи	Выключает компьютер
помощь	Показывает список команд
стоп	Выключает Джарвиса
